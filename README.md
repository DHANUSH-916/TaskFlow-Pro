# 📋 TaskFlow Pro

A Full Stack MERN Task Management Application built using React, Node.js, Express, and MongoDB.

## 🚀 Features

- User Registration
- User Login (JWT Authentication)
- Dashboard
- Create Tasks
- Edit Tasks
- Delete Tasks
- Complete Tasks
- Search Tasks
- Filter by Status
- Filter by Priority
- Sort Tasks
- Responsive UI

## 🛠 Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Bootstrap
- React Toastify

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/DHANUSH-916/TaskFlow-Pro.git
```

### Backend

```bash
cd server
npm install
node server.js
```

### Frontend

```bash
npm install
npm run dev
```
